# なかよしわりかん - 画面設計

## 画面一覧

1. **メイン画面** - メンバー・会計の一覧と操作
2. **会計追加シート** - ボトムシートで会計を追加/編集
3. **精算結果画面** - 誰が誰にいくら払うか表示
4. **設定シート** - 通貨・端数単位の設定

## メイン画面

```
┌─────────────────────────────────┐
│  なかよしわりかん        ⚙️    │
├─────────────────────────────────┤
│  メンバー                       │
│  [太郎] [花子] [次郎] [+]       │
├─────────────────────────────────┤
│  会計                           │
│  ┌─────────────────────────────┐│
│  │ 1次会  ¥15,000              ││
│  │ 立替: 太郎 / 参加: 全員     ││
│  └─────────────────────────────┘│
│  [+ 会計を追加]                 │
├─────────────────────────────────┤
│  [💰 精算結果を見る]            │
└─────────────────────────────────┘
```

## 会計追加シート

- 名前入力
- 電卓 UI で金額入力
- 立替者選択（ラジオ）
- 参加者選択（チェックボックス）

## 精算結果画面

- 合計金額・1 人あたり表示
- 支払いフロー一覧
- 内訳表示
- 結果をコピーボタン

## 設定シート

- 通貨選択
- 端数単位（1 円 / 10 円 / 100 円）
- データリセット
